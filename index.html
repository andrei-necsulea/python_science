What is Python ?

         Python is a high-level, interpreted, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation.


         Python is dynamically-typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly procedural), object-oriented and functional programming. It is often described as a "batteries included" language due to its comprehensive standard library. Guido van Rossum began working on Python in the late 1980s as a successor to the ABC programming language and first released it in 1991 as Python 0.9.0 Python 2.0 was released in 2000 and introduced new features such as list comprehensions, cycle-detecting garbage collection, reference counting, and Unicode support. Python 3.0, released in 2008, was a major revision that is not completely backward-compatible with earlier versions.


Python 2 was discontinued with version 2.7.18 in 2020.
Python consistently ranks as one of the most popular programming languages.





How to install Python ?

      For Windows users :                

Step 1 :

Download python from the official site: www.python.org . I personally recommend to install version 3.10.4 because you can use all the improvments , security fixes and bugs-free. I have to mention that is mandatory to have at least Windows 10 to run this verison of python. However, you can choose an older version of python, or make a virtual machine with Windows 10 and run python. Before go further , I reccomend you to mark "Add python to path"(More info here) meanwhile the installation . This is for global system variables , which basically means that you want Windows recognize Python and let it run.(All info here)

               

Step 2 :

Download Visual Studio Code(Official Site here). After installing it , you will need some extensions for running python. First one on the list is CodeRunner - run code in terminal(Set instructions). It is very important to know that for every module you wanna use , you must install it manually in CMD(Command Prompt , without any administrator privilegies). Command for installing : "pip install module".
