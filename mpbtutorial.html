What is Matplotlib?

    Matplotlib is a low level graph plotting library in python that serves as a visualization utility.
    Matplotlib was created by John D. Hunter.
    Matplotlib is open source and we can use it freely.
    Matplotlib is mostly written in python, a few segments are written in C, Objective-C and Javascript for Platform compatibility.






What can be done with Matplotlib?

    With matplotlib you could use a large variety of plots which can be customized in many different ways.
    In this tutorial we are going to cover some sample type of plots (mostly the frequently used of them).



Do you really need Python at a certain level for using Matplotlib?

    Yes , you have to know the basics of python and linear algebra. However , it's not necessary to be an expert of python if you don't search a job into data science field. If you wanna use it for any purpose which not involve data science field , firstly learn basics of python and then start matplotlib.
